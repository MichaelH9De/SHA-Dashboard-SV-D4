# Social Value Platform

A rebuilt Next.js scaffold for managing framework-led social value delivery.

## Included in this version
- Premium dashboard and page layout
- Seed data for projects, measures, frameworks, evidence and reports
- Reusable UI kit for cards, tables, badges, tabs and form surfaces
- Static API routes returning useful JSON
- Notes and docs to guide next implementation steps

## Run locally

```bash
npm install
npm run dev
```
